## 项目简介

本仓库用于整理商业软件系统、移动应用、多人在线游戏以及企业级解决方案的技术资料与架构案例。

主要面向：

* 软件开发工程师
* 创业团队
* 产品经理
* 技术负责人
* 系统架构师

通过研究不同项目案例，可以了解商业软件系统的设计思路、架构方案以及功能模块规划。
